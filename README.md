# runegen #

Random string generator.

### What is this repository for? ###

* Quick summary
* Version 1.0.0

### How do I get set up? ###

* import the library in go lang code:
  
```bash
$# go get github.com/berrypay/runegen
```

```go
import (
  "fmt"
  "github.com/berrypay/runegen"
)

func main() {
  fmt.Println(runegen.GetRandom(1, 6))
}
```
